# DSAI4103-Churn-Project-60106354
## Project Title
Telecom Customer Churn Prediction

## Business Problem
This project aims to predict customer churn in a telecom company and identify the factors that influence whether a customer leaves the service.

## Dataset
The project uses multiple data sources:
- billing.xlsx
- customers.xlsx
- services.xlsx
- Telco_customer_churn.xlsx

## Project Contents
- Jupyter Notebook for preprocessing, EDA, and predictive modeling
- Tableau dashboard for interactive visualization
- Packaged model file

## Dashboard
The dashboard includes:
- Churn Distribution
- Contract vs Churn
- Tenure vs Churn
- Monthly Charges vs Tenure
- Customer Churn by Internet Service Type

## Tools Used
- Python
- Jupyter Notebook
- Tableau
- Scikit-learn
- SHAP
